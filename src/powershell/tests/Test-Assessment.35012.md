Container labels extend sensitivity classification beyond individual files to entire collaboration workspaces like Microsoft Teams, Microsoft 365 Groups, and SharePoint sites. These labels control workspace-level settings such as external sharing, guest access, device restrictions, and privacy.

Without container labels, users can create Teams with external guest access even when handling confidential information, creating data exfiltration risks where properly labeled documents exist in improperly secured workspaces. Container labels ensure workspace security matches the sensitivity of stored content, preventing "Highly Confidential" documents from residing in Teams that permit external sharing.

**Remediation action**

- [Use sensitivity labels to protect content in Microsoft Teams, Microsoft 365 groups, and SharePoint sites](https://learn.microsoft.com/purview/sensitivity-labels-teams-groups-sites?wt.mc_id=zerotrustrecommendations_automation_content_cnl_csasci)
