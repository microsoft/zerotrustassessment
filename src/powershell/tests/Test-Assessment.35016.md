Without mandatory labeling, users can share unclassified content, creating security and compliance risks. Threat actors can exfiltrate sensitive data without classification metadata to trigger protection policies.

If DLP policies rely on label detection, unclassified data bypasses these controls entirely. Organizations should enable mandatory labeling across all workloads to ensure communications, documents, and analytics content are classified before sharing.

**Remediation action**

- [Publish sensitivity labels](https://learn.microsoft.com/purview/create-sensitivity-labels?tabs=modern-label-scheme&wt.mc_id=zerotrustrecommendations_automation_content_cnl_csasci#publish-sensitivity-labels-by-creating-a-label-policy) [Requiring users to apply a label](https://learn.microsoft.com/purview/sensitivity-labels-office-apps?wt.mc_id=zerotrustrecommendations_automation_content_cnl_csasci#require-users-to-apply-a-label-to-their-email-and-documents).
