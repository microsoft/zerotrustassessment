Without justification requirements, users can silently downgrade labels without creating an audit trail, creating compliance risk. For example, when a user removes a "Confidential" label and replaces it with "Internal," organizations should require explicit justification.

Without this control, compromised accounts or departing employees could downgrade labels to enable data exfiltration. Downgrade justification is a lightweight control that increases accountability without impacting user workflows.
